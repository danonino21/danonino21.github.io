Proyecto destacado

01

Football Oracle — agente de IA para predicción futbolística

↗ github.com/danonino21/football-oracle

Agente conversacional con herramientas propias para ingesta, análisis y proyección de resultados de la Primera División chilena. El agente decide de forma autónoma qué herramienta invocar — scraper, motor estadístico o API externa — según el objetivo del usuario, con razonamiento paso a paso y salida en lenguaje natural.

Python LangChain LangGraph Flask · API REST Playwright Dixon-Coles Monte Carlo
  • Orquestación multi-step con LangGraph: el agente selecciona herramientas sin intervención manual, incluyendo decisiones de manejo de errores en la integración entre sistemas.
  • Scraper con Playwright contra Sofascore para extraer xG y estadísticas avanzadas, sorteando bloqueos de Cloudflare.
  • Modelo estadístico Dixon-Coles + simulación Monte Carlo para proyectar probabilidades de campeonato.
  • Sistema en producción, consultable vía API REST Flask con dashboard interactivo (predicciones, tabla de posiciones, chat con el agente).
